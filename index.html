Markup Language


HTML

Hypertext Markup Language (HTML) adalah bahasa markup standar yang dirancang untuk ditampilkan di browser. HTML dapat dibantu dengan teknologi seperti CSS dan bahasa scripting seperti JavaScript. Semua website di Internet dibangun dengan HTML.

Styling Language


CSS

CSS (Cascading Style Sheets) dipakai untuk mendesain halaman depan atau tampilan website (front end). CSS menangani tampilan dan ‘rasa’ dari halaman website. Semua website modern sudah pasti menggunakan CSS.

CSS Framework


BootStrap

Bootstrap adalah framework CSS untuk merancang situs web dan aplikasi web. Framework ini berisi templat desain berbasis HTML dan CSS untuk tipografi, formulir, tombol, navigasi, dan komponen antarmuka lainnya, serta juga ekstensi opsional JavaScript.

Programming Language


C++

C++ adalah bahasa pemrograman yang dibuat oleh Bjarne Stroustrup, yang merupakan perkembangan dari bahasa C dikembangkan di Bell Labs (Dennis Ritchie). C++ akan kita gunakan untuk memahami konsep dasar dari pemrograman saja. Jika kamu sudah memahami dasar-dasar pemrograman sebelumnya, kamu bisa skip bagian ini.

JavaScript

JavaScript (JS) adalah bahasa pemrograman tingkat tinggi, dinamis, multi-paradigm, event-driven, non-blocking, dan asynchronous yang dapat berjalan di front-end maupun back-end.

Runtime Environment


Node.js

Node.js adalah runtime environment yang digunakan untuk menjalankan source code JavaScript di server layaknya bahasa pemrograman back-end lainnya. Contoh website yang dibangun menggunakan Node.js yaitu: medium.com, imdb.com, dan website ini.

Back-End Framework


Express

Express adalah back-end framework untuk Node.js. Express dirancang untuk membangun aplikasi web dan API. Express merupakan framework server standar untuk Node.js. Contoh website yang dibangun menggunakan Express yaitu: paypal.com, shopee.co.id

Front-End Library


React

React adalah front-end JavaScript library untuk membangun user interface yang berdasarkan UI component. React dapat digunakan sebagai basis dalam pengembangan single-page app atau mobile app. Contoh website yang dibangun menggunakan React yaitu: facebook.com, tokopedia.com, dan website ini.

Programming Language


TypeScript

TypeScript adalah sebuah bahasa pemrograman yang merupakan superset dari JavaScript. TypeScript membawa fitur seperti IntelliSense yang lebih predictable, fitur OOP yang lebih advanced, dan berkat perilakunya yang static typed, dapat membantu proses debugging menjadi lebih mudah. Text Editor terkenal yaitu VSCode dibangun menggunakan TypeScript.
